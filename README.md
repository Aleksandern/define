# DeFine

![Status](https://img.shields.io/badge/status-active%20development-blue)
![Stack](https://img.shields.io/badge/stack-typescript%20%7C%20evm-black)

DeFi risk observability and early warnings for EVM blockchains.

## Overview

DeFine is a DeFi risk observability system focused on early risk signals, security warnings, and explainable alerts for wallet exposure across EVM protocols.

The goal is to convert raw on-chain state into deterministic risk signals that help users understand and respond to risks in their DeFi activity.

## Status

Work in progress.

Parts of the risk evaluation modules will be released as open-source libraries.
